# Webpack概念

Webpack 的核心概念是一个 模块打包工具 ，它的主要目标是将js文件打包在一起，打包后的文件用于在目标环境中使用，但它也能胜任 转换transform 、打包bundle 或 包裹package 任何其他资源。
